# VN_LCD Website

Thư mục này là website tĩnh giới thiệu thư viện `VN_LCD`, sẵn sàng deploy bằng Cloudflare Pages.

## Cloudflare Pages

- Framework preset: `None`
- Build command: để trống
- Build output directory: `site`
- Root directory: thư mục repo chứa `site`

## SEO cần đổi khi có domain thật

Trong các file sau, đổi `https://vn-lcd.pages.dev/` thành domain Cloudflare Pages hoặc domain riêng của bạn:

- `index.html`
- `robots.txt`
- `sitemap.xml`

Các cụm từ khóa chính đã được đặt trong title, description, nội dung trang và JSON-LD:

- `thư viện arduino LCD tiếng việt`
- `in chữ tiếng Việt trên LCD bằng arduino`
- `LCD 16x2 I2C HD44780 PCF8574`
